Stimulation Event List

The first step fot the Stimic project is to list all the stimulations, their localization, the electrical parameters, and associate each stimulation with its potential effect on the iEEG

To gain speed, a semi-automatic detection pipeline was developped. The different steps are :

  1. Clean the different EDF files, keep only good EEG channels
  2. Run the stimulation detector
  3. Visual check of the event detected using micmac and export the corrected event list.
  4. Re-organized the csv file (in Excel) and add a bipolar channel column
  5. Concatenate these event files for all edf files

All the procedure is described here)
